# bl-readings-service

## Starting Spark

```bash
mvn scala:run -DmainClass=org.jesperancinha.bridgelogistics.readings.ReadingsLauncher
```

## Installing cassandra

```bash
brew install cassandra
```

## Starting cassandra

```bash
cassandra
```

## Cassandra command line

```bash
cqlsh
```

```genericsql
DESCRIBE TABLES
SELECT *  FROM java_api.products;

```


## Hints & Tricks

```bash
brew install sbt
brew upgrade sbt
```
